Minecraft

Currently playing minecraft. Hosting a server locally and a friend and I play. Using Neoforge and a bunch of cleint plugins and shaders. A few server plugins, one for performance, one for discord integration. Client side I'm using Prism Launcher on my "gaming pc" which is pretty modest but more than enough for minecraft. I suppose I should say that non java minecraft is the devil or something but I know too many kids who play bedrock or pocket to be that much of a jerk. I would say that I prefer java.

I run the server via docker compose -well, portainer stacks, using itzg/docker-minecraft-server a la: 

services:
  mc:
    image: itzg/minecraft-server
    container_name: mc-server-20241012
    tty: true
    stdin_open: true
    restart: unless-stopped
    ports:
      - "25565:25565" 
    environment:
      EULA: "TRUE"
      EXEC_DIRECTLY: "true"
      TZ: ""
      SNOOPER_ENABLED: "false"
      MEMORY: "8G"
      TYPE: "NEOFORGE"                      
      NEOFORGE_VERSION: "latest"        
      VERSION: "1.21.1"
      MODE: "SURVIVAL"
      FORCE_GAMEMODE: "true"
      DIFFICULTY: "HARD"
      ANNOUNCE_PLAYER_ACHIEVEMENTS: "true"
      PVP: "false"      
      OPS: "" 
      LEVEL: ""
      LEVEL_SEED: "1975020419760203"

(I've removed some details) Of note: I first tried seting some server.properties in the compose but ended up having to manually add a few files -whitelist, ops, and manually edit server.properties.
